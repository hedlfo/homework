2장 네트워크 연결과 구성요소

네트워크 연결구분
LAN(Local Area Network) : 비교적 소규모의 네트워크, 현재 대부분 이더넷 기반 전송 기술 사용.

WAN(Wide Area Network) : 멀리 떨어진 LAN을 연결해주는 네트워크
먼 거리에 있는 네트워크를 연결하기 위해 사용됨,
특별한 경우가 아니면 직접 구축할 수 없는 범위의 네트워크이므로 대부분 통신사업자로부터 
회선을 임대해 사용.

네트워크 회선
1. 인터넷 회선
인터넷 접속을 위해 통신사업자와 연결하는 회선
일반 인터넷 회선의 종류-
-광랜(이더넷)
-FTTH
-동축 케이블 인터넷
-xDSL(ADSL, VDSL등)

2. 전용 회선
가입자와 통신사업자 간에 대역폭을 보장해주는 서비스
가입자와 통신사업자 간은 전용 케이블로 연결되어 있고 통신사업자 내부에서
TDM(시분할 다중화)같은 기술로 마치 직접 연결한 것처럼 통신 품질을 보장.

3. 인터넷 전용 회선
인터넷 연결 회선에 대한 통신 대역폭을 보장해주는 상품
가입자가 통신사업자와 연결되고 이 연결이 다시 인터넷과 연결되는 구조
통신사업자와 가입자 간 연결 품질을 보장해줌.
최근 인터넷 전용 회신 기술은 이더넷이 가장 많이 쓰임.

4. VPN
물리적으로는 전용선이 아니지만 가상으로 직접 연결한 것 같은 효과가 나도록 
만들어주는 네트워크 기술.
-통신사업자 VPN
-가입자 VPN

5.DWDM
먼거리를 통신할 때 케이블 포설 비용이 매우 많이 들고 관리가 어려운 문제를
극복하기 위해 개발됨.
하나의 광케이블에 다른 파장의 빛을 통해 여러 채널을 만드는 동시에 많은 데이터를
전송할 수 있음.
기가 인터넷은 FTTH가 사용되고 구축 방식에 따라 PTP, AON, PON 형태로 구분됨.
PTP: 가입자와 통신사업자 간에 케이블을 직접 포설
AON : 광신호 분리장비에 정기가 필요한 스위치와 같은 장비가 사용
PON: 전기 인입 없이 광신호를 분리해 가입자와 통신사업자 간의 케이블을 줄일 수 있음.


네트워크 구성 요소

1. 네트워크 인터페이스 카드(NIC)
:컴퓨터를 네트워크에 연결하기 위한 하드웨어 장치
주요 역할 1-직렬화(전기신호 형태로 데이터 전송)
2-MAC 주소(받은 패킷의 돋착지 주소가 자신의 MAC주소가 아니면 폐기, 맞으면 전달)
3-흐름 제어(데이터 유실 방지를 위해 데이터를 받지 못할 때는 상대방에게 통신 중지 요청)

2. 케이블과 커넥터
1-트위스티드 페어 케이블
가장 흔히 사용하는 케이블.
STP-그물 형태의 쉴드가 있는것
FTP-포일 형태의 쉴드가 있는것
UTP-쉴드가 없는 것
2-동축 케이블
케이블 TV와 연결할 때 사용하는 두꺼운 검정 케이블.
DAC케이블이 이에 속함
3-광케이블
다른 구리선보다 신뢰도가 높고 더 먼거리까지 통신할 수 있어 높은 대역폭을 요구하거나
먼 거리를 통신해야 하는 네트워크 장비 간의 통신에 주로 사용됨.
광케이블은 싱글모드(먼 거리 통신을 지원위해 케이블 굵기가 가늘고 광원으로 레이저 사용)
와 멀티모드(싱글모드에 비해 비교적 굵은 케이블을 사용하며 광원으로 LED사용)
싱글모드는 노란색, 멀티모드는 주황색과 하늘색을 띔.
4-커넥터
케이블의 끝부분으로 네크워크 장비나 네트워크 카드에 연결되는 부분
5-트랜시버
다양한 외부 신호를 컴퓨터 내부의 전기 신호로 바꾸어줌.

3. 허브
케이블과 동일한 1계층에서 동작하는 장비 현재는 거의 사용하지 않음.

4. 스위치
여러 장비를 연결하고 통신을 중재하는 2계층 장비.
스위칭 허브라고도 함
허브는 단순히 전기 신호를 재생성하여 모든 포트에 전기 신호를 보내지만 스위치는 허브와
달리 MAC 주소를 이해하여 목적지 MAC주소의 위치를 파악하고 목적지가 연결된 포트로만 
전기 신호를 보냄.

5. 라우터
점점 필요성이 증가되는 것. 3계층에서 동작하며 먼거리로 통신할 수 있는 프로토콜로 변환.


6. 로드 밸런서
일반적으로 4계층에서 동작하며 4계층 포트 주소를 확인하는 동시에 IP 주소를 변경할 수 
있음.

7. 보안 장비
보안장비는 네트워크 장비와 달리 정보를 잘 제어하고 공격을 방어하는데 초점이 맞추어져 있음.

8. 기타(모뎀/공유기 등)
모뎀- 짧은 거리를 통신하는 기술과 먼 거리를 통신하는 기술이 달라 이 기술들을 변환해 주는 장비
